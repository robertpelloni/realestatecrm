## What this is

This repository is the planning and operational source of truth for a premium CRM that combines practical strengths of platforms like Lofty and MyRealSource, while adding:

- AI lead qualification and follow-up
- conversational voice workflows
- omnichannel communication and social publishing
- client portals and deal tracking
- document exchange and e-signatures
- marketing studio outputs
- partner modules for mortgage, title, insurance, and other adjacent businesses
- web + mobile parity with synced state

## Design priorities

- Luxury black / blue / gold visual identity
- Highly readable text and accessible contrast
- Role-based permissions for broker, agent, admin, client, and partner users
- Extensible workflow engine with visible Add Workflow / Add Step controls
- TypeScript for both the frontend and backend (Next.js 15 App Router)
- Reliable mobile behavior with offline-friendly sync and recovery
