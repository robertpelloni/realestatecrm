- Authentication and roles
- Workspace model

- Unified inbox
- Publishing calendar
- Marketing studio
- Asset export flows
- Approval workflows

## Phase 5 — Media pipeline

- Render/export jobs

- Mortgage / title / insurance modules

- Partner permissions

- Advanced import/export
- Performance hardening

## Phase 7 — Polish and reliability
- Mobile offline support
- Sync recovery
- Audit improvements
- Accessibility refinement
- UI polish
