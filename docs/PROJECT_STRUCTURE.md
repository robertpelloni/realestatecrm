# Project Structure & Libraries

## Directory Layout (Planned)

```
.
├── docs/                 # Project documentation (PRD, Tech Spec, Roadmap)
├── src/
│   ├── app/              # Next.js App Router (Frontend + Backend APIs)
│   ├── components/       # Reusable UI components
│   ├── lib/              # Utility functions, API clients, shared logic
│   ├── styles/           # Global styles and theme definitions
│   └── types/            # Global TypeScript interfaces and types
├── public/               # Static assets
├── tests/                # Unit and integration tests
└── [Config Files]        # package.json, tsconfig.json, etc.
```

## Submodules & Libraries (Planned)

- **Framework:** Next.js (React)
- **Language:** TypeScript
- **Styling:** Tailwind CSS (configured for luxury Black/Blue/Gold theme)
- **Database ORM:** Prisma / Drizzle (TBD based on infrastructure)
- **Authentication:** NextAuth.js or Supabase Auth
- **State Management:** Zustand or React Context

_Note: This file will be continuously updated as submodules and specific package versions are installed._
